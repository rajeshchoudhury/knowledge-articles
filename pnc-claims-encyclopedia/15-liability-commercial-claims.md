## 1. General Liability (CGL) Policy Structure

### 1.1 CGL Coverage Parts

```
  COMMERCIAL GENERAL LIABILITY (CGL) POLICY
  ═════════════════════════════════════════

  ┌──────────────────────────────────────────────────────────────┐
  │ COVERAGE A: BODILY INJURY AND PROPERTY DAMAGE LIABILITY      │
  │                                                               │
  │ Insuring Agreement:                                          │
  │ "We will pay those sums that the insured becomes legally     │
  │  obligated to pay as damages because of 'bodily injury' or  │
  │  'property damage' to which this insurance applies."         │
  │                                                               │
  │ Includes:                                                    │
  │ • Premises liability (slip-and-fall, inadequate security)    │
  │ • Operations liability (damage during business operations)   │
  │ • Products liability (injury from product sold)              │
  │ • Completed operations (damage after work is complete)       │
  │ • Contractual liability (assumed under contract)             │
  │                                                               │
  │ Key Limits:                                                  │
  │ • Each Occurrence Limit:     $1,000,000 (typical)           │
  │ • General Aggregate Limit:    $2,000,000 (typical)           │
  │ • Products-Completed Ops Aggregate: $2,000,000 (typical)     │
  │                                                               │
  │ Duty to Defend: YES (broader than duty to indemnify)        │
  └──────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────┐
  │ COVERAGE B: PERSONAL AND ADVERTISING INJURY LIABILITY        │
  │                                                               │
  │ Covers offenses such as:                                     │
  │ • False arrest, detention, or imprisonment                   │
  │ • Malicious prosecution                                      │
  │ • Wrongful eviction or entry                                 │
  │ • Oral or written defamation (slander/libel)                │
  │ • Violation of right of privacy                              │
  │ • Copyright infringement in advertising                      │
  │ • Misappropriation of advertising ideas                      │
  │                                                               │
  │ Key Limits:                                                  │
  │ • Each Offense Limit:        $1,000,000 (typical)           │
  │ • Subject to General Aggregate                               │
  └──────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────┐
  │ COVERAGE C: MEDICAL PAYMENTS                                 │
  │                                                               │
  │ Covers medical expenses for persons injured on or because    │
  │ of insured's premises or operations, regardless of fault.    │
  │                                                               │
  │ Key Limits:                                                  │
  │ • Per Person Limit:          $5,000 - $10,000 (typical)     │
  │ • No fault required                                          │
  │ • Voluntary / goodwill coverage                              │
  └──────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────┐
  │ SUPPLEMENTARY PAYMENTS (in addition to limits)               │
  │                                                               │
  │ • Defense costs (outside of limits in most CGL)              │
  │ • Bail bonds (up to $250)                                    │
  │ • Appeal bonds                                               │
  │ • Loss of earnings ($250/day for court attendance)           │
  │ • Reasonable expenses incurred at carrier request             │
  │ • Post-judgment interest                                     │
  │ • Pre-judgment interest (on indemnity portion)               │
  └──────────────────────────────────────────────────────────────┘
```

## 2. Occurrence vs Claims-Made

### 2.1 Coverage Trigger Comparison

```
  OCCURRENCE vs CLAIMS-MADE COVERAGE
  ═══════════════════════════════════

  OCCURRENCE FORM:
  ┌──────────────────────────────────────────────────────────┐
  │ Trigger: When the INJURY/DAMAGE OCCURS                   │
  │                                                          │
  │ Timeline:                                                 │
  │ ├──────────┤                      ├──────────┤           │
  │ Policy     Policy                 Claim      Settlement  │
  │ Effective  Expired                Reported   Paid        │
  │ 01/01/24   12/31/24              03/15/26   09/15/27    │
  │                                                          │
  │       ●──── Injury occurs 06/15/24 ────●                │
  │                                                          │
  │ COVERED: Injury occurred during policy period,           │
  │ regardless of when claim is reported                     │
  │                                                          │
  │ Best for: CGL, most commercial liability                 │
  │ "Tail" is built in (perpetual reporting)                 │
  └──────────────────────────────────────────────────────────┘

  CLAIMS-MADE FORM:
  ┌──────────────────────────────────────────────────────────┐
  │ Trigger: When the CLAIM IS FIRST MADE (reported)         │
  │ AND the wrongful act occurred after the retroactive date │
  │                                                          │
  │ Timeline:                                                 │
  │ ├──────────┤├──────────┤                                 │
  │ Retro     Policy     Policy                              │
  │ Date      Effective  Expired                              │
  │ 01/01/20  01/01/24   12/31/24                            │
  │                                                          │
  │      ●─── Act occurs 03/15/22 ───●                      │
  │                                                          │
  │           ●─── Claim made 06/15/24 ───●                  │
  │                                                          │
  │ COVERED: Act occurred after retro date AND               │
  │ claim was made during policy period                      │
  │                                                          │
  │ NOT COVERED if:                                          │
  │ • Claim made after policy expires (without tail/ERP)     │
  │ • Act occurred before retroactive date                   │
  │                                                          │
  │ Best for: E&O, D&O, EPLI, Cyber, Medical Malpractice   │
  │ Requires "tail" (ERP) when switching carriers            │
  └──────────────────────────────────────────────────────────┘

  EXTENDED REPORTING PERIOD (ERP / "Tail"):
  ┌──────────────────────────────────────────────────────────┐
  │ Basic ERP: 60 days (automatic, no extra charge)          │
  │ Supplemental ERP: 1-5 years (purchased, typically        │
  │   100-200% of annual premium for unlimited tail)         │
  │ Mini-Tail: 1-3 years at 75-150% of premium              │
  └──────────────────────────────────────────────────────────┘
```

### 2.2 Coverage Trigger Matrix by Line

| Line of Business | Common Form | Trigger |
|---|---|---|
| CGL | Occurrence | When injury/damage occurs |
| Professional Liability (E&O) | Claims-Made | When claim is reported |
| D&O | Claims-Made | When claim is reported |
| EPLI | Claims-Made | When claim is reported |
| Cyber | Claims-Made | When claim is reported |
| Medical Malpractice | Claims-Made (some Occurrence) | When claim is reported |
| Environmental | Claims-Made | When claim is reported |
| Product Liability (within CGL) | Occurrence | When injury occurs |
| Excess/Umbrella | Follows underlying | Matches primary |

### 3.2 Products Liability

| Theory | Description | Burden on Plaintiff |
|---|---|---|
| Manufacturing Defect | Specific unit deviates from design | Product deviated from intended design |
| Design Defect | Entire product line is unreasonably dangerous | Alternative design was feasible and safer |
| Failure to Warn | Inadequate warnings or instructions | Adequate warning would have prevented injury |
| Breach of Warranty | Express or implied warranty violated | Product failed to meet warranty |
| Strict Liability | Liability without proof of negligence | Product was defective and caused injury |

## 5. Liability Investigation Process

### 5.1 Investigation Framework

```
  LIABILITY CLAIM INVESTIGATION PHASES
  ═════════════════════════════════════

  Phase 1: INITIAL ASSESSMENT (Day 1-3)
  ┌──────────────────────────────────────────────────────────┐
  │ □ Review FNOL details and coverage in force              │
  │ □ Assess severity and exposure (initial reserve)         │
  │ □ Identify coverage issues (ROR needed?)                 │
  │ □ Determine investigation needs                          │
  │ □ Preserve evidence (video, photos, physical)            │
  │ □ Issue litigation hold if suit filed                    │
  │ □ Contact insured for initial statement                  │
  │ □ Assign defense counsel if suit filed                   │
  │ □ Report to excess carrier if applicable                 │
  │ □ Check for additional insured requests                  │
  └──────────────────────────────────────────────────────────┘

  Phase 2: COVERAGE ANALYSIS (Day 1-15)
  ┌──────────────────────────────────────────────────────────┐
  │ □ Verify policy in force at time of occurrence           │
  │ □ Identify applicable coverage parts (A, B, C)          │
  │ □ Analyze exclusions                                     │
  │ □ Determine occurrence vs claims-made trigger            │
  │ □ Review additional insured endorsements                 │
  │ □ Analyze other insurance provisions                     │
  │ □ Review contractual obligations                         │
  │ □ Issue Reservation of Rights letter (if needed)         │
  │ □ Document coverage position                             │
  └──────────────────────────────────────────────────────────┘

  Phase 3: LIABILITY INVESTIGATION (Day 3-60)
  ┌──────────────────────────────────────────────────────────┐
  │ □ Obtain police/incident reports                         │
  │ □ Take recorded statements (insured, witnesses)          │
  │ □ Inspect premises / occurrence scene                    │
  │ □ Photograph and document scene                          │
  │ □ Obtain surveillance video                              │
  │ □ Review maintenance records, inspection reports         │
  │ □ Research applicable standards of care                  │
  │ □ Retain experts if needed (engineering, medical, etc.)  │
  │ □ Obtain contracts, certificates of insurance            │
  │ □ Determine proximate cause                              │
  │ □ Assess comparative/contributory negligence             │
  │ □ Identify responsible parties and contribution rights   │
  └──────────────────────────────────────────────────────────┘

  Phase 4: DAMAGES EVALUATION (Day 15-90)
  ┌──────────────────────────────────────────────────────────┐
  │ □ Obtain and review medical records (if BI)              │
  │ □ Verify medical treatment and causation                 │
  │ □ Evaluate lost wages / earning capacity                 │
  │ □ Obtain property damage estimates                       │
  │ □ Assess general damages (pain/suffering)                │
  │ □ Research comparable verdicts in jurisdiction            │
  │ □ Calculate total exposure (best/worst/likely)           │
  │ □ Evaluate coverage adequacy vs exposure                 │
  │ □ Adjust reserves to current evaluation                  │
  └──────────────────────────────────────────────────────────┘

  Phase 5: RESOLUTION (Day 30-365+)
  ┌──────────────────────────────────────────────────────────┐
  │ □ Determine settlement authority                         │
  │ □ Negotiate with claimant/attorney                       │
  │ □ Mediate if appropriate                                 │
  │ □ Prepare for trial if not settling                      │
  │ □ Execute settlement / obtain release                    │
  │ □ Process payments                                       │
  │ □ Pursue contribution from other parties                 │
  │ □ Close claim file                                       │
  └──────────────────────────────────────────────────────────┘
```

## 6. Coverage Analysis

### 6.1 Duty to Defend vs Duty to Indemnify

```
  DUTY TO DEFEND vs DUTY TO INDEMNIFY
  ═══════════════════════════════════

  DUTY TO DEFEND:
  ┌──────────────────────────────────────────────────────────┐
  │ • Broader than duty to indemnify                         │
  │ • Triggered by ALLEGATIONS in complaint                  │
  │ • If ANY allegation POTENTIALLY falls within coverage,   │
  │   insurer must defend the ENTIRE action                  │
  │ • "Four corners" test: compare complaint to policy       │
  │ • Some jurisdictions allow extrinsic evidence            │
  │ • Defense costs paid in addition to limits (CGL)         │
  │ • Insurer selects and pays defense counsel               │
  └──────────────────────────────────────────────────────────┘

  DUTY TO INDEMNIFY:
  ┌──────────────────────────────────────────────────────────┐
  │ • Narrower than duty to defend                           │
  │ • Determined by ACTUAL FACTS (not just allegations)      │
  │ • Applies only if damages actually fall within coverage  │
  │ • May exist with or without duty to defend               │
  │ • Determined at conclusion of underlying action          │
  └──────────────────────────────────────────────────────────┘

  DECISION FLOW:
  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │  Read complaint ──▶ Compare to policy ──▶ Potentially    │
  │  allegations         coverage terms        covered?      │
  │                                              │           │
  │                                      YES     │     NO    │
  │                                      │       │     │     │
  │                                      ▼       │     ▼     │
  │                                   DEFEND     │  NO DUTY  │
  │                                   (+ ROR if  │  TO       │
  │                                    coverage  │  DEFEND   │
  │                                    uncertain)│           │
  │                                              │           │
  │                                              ▼           │
  │                                       INDEMNIFY?         │
  │                                       (based on actual   │
  │                                        facts at trial)   │
  └──────────────────────────────────────────────────────────┘
```

### 6.2 Additional Insured Analysis

```
  ADDITIONAL INSURED COVERAGE ANALYSIS
  ═════════════════════════════════════

  Step 1: Is there an Additional Insured endorsement?
  ├── Check CG 20 10 (Ongoing Operations)
  ├── Check CG 20 37 (Completed Operations)
  ├── Check CG 20 33 (Owners, Lessees, Contractors - Automatic)
  └── Check manuscript endorsements

  Step 2: Does the requesting party qualify?
  ├── Named specifically on endorsement?
  ├── Or qualify under blanket "any person or organization"
  │   "required by written contract"?
  └── Is there a written contract requiring AI status?

  Step 3: Is the claim within scope of AI coverage?
  ├── CG 20 10 (2004+): Only for liability caused IN WHOLE
  │   OR IN PART by named insured's acts/omissions
  ├── CG 20 10 (pre-2004): Broader – includes vicarious
  │   liability of additional insured
  └── Does claim arise from named insured's work?

  Step 4: Apply Other Insurance provisions
  ├── Primary and non-contributory? (if endorsed)
  ├── Or excess over AI's own insurance?
  └── Pro-rata / contribution by equal shares?
```

## 7. Reservation of Rights Letters

### 7.1 When to Issue ROR

| Situation | ROR Needed | Urgency |
|---|---|---|
| Potential exclusion applies | Yes | High - issue within 7-15 days |
| Late notice defense | Yes | High |
| Allegations outside coverage | Yes | High |
| Multiple potential coverage defenses | Yes | High |
| Policy limits may be inadequate | Yes (excess letter) | Medium |
| No coverage issues identified | No | N/A |
| Clear coverage denial | No (denial letter instead) | High |

### 7.2 ROR Letter Components

```
  RESERVATION OF RIGHTS LETTER CONTENT
  ═════════════════════════════════════

  ┌──────────────────────────────────────────────────────────┐
  │ 1. IDENTIFICATION                                        │
  │    • Claim number and date of loss                       │
  │    • Policy number and policy period                     │
  │    • Named insured                                       │
  │    • Claimant / plaintiff name                           │
  │    • Lawsuit caption and case number (if suit filed)     │
  │                                                          │
  │ 2. FACTS AS UNDERSTOOD                                   │
  │    • Summary of occurrence as reported                   │
  │    • Allegations in complaint (if suit)                  │
  │                                                          │
  │ 3. COVERAGE CONCERNS                                     │
  │    • Specific policy provisions at issue                 │
  │    • Each exclusion or condition that may apply          │
  │    • Quote relevant policy language verbatim             │
  │    • Explain how each provision may apply to facts       │
  │                                                          │
  │ 4. RESERVATION LANGUAGE                                  │
  │    • "Nothing in this letter should be construed as a    │
  │      waiver of any right, defense, or coverage position" │
  │    • Reserve right to deny coverage after investigation  │
  │    • Reserve right to withdraw defense if no coverage    │
  │    • Reserve right to seek reimbursement of defense costs│
  │      (in applicable jurisdictions)                       │
  │                                                          │
  │ 5. DEFENSE PROVISION                                     │
  │    • Despite coverage concerns, company will provide     │
  │      defense (under reservation of rights)               │
  │    • Defense counsel identification                      │
  │    • Insured's right to retain own counsel at own expense│
  │    • In some jurisdictions: right to independent counsel │
  │      at carrier's expense (e.g., California - San Diego  │
```

## 10. Excess and Umbrella Coverage

### 10.1 Excess vs Umbrella

```
  EXCESS vs UMBRELLA COVERAGE
  ═══════════════════════════

  FOLLOWING-FORM EXCESS:
  ┌──────────────────────────────────────────────────────────┐
  │ • Follows terms and conditions of underlying policy      │
  │ • No broader coverage than underlying                    │
  │ • Pure dollar extension of limits                        │
  │ • No drop-down provision                                 │
  │                                                          │
  │ Example:                                                 │
  │ Primary CGL: $1M per occurrence                          │
  │ Excess: $4M xs $1M                                       │
  │ → Excess pays $1M to $5M (same terms as CGL)            │
  └──────────────────────────────────────────────────────────┘

  UMBRELLA:
  ┌──────────────────────────────────────────────────────────┐
  │ • May be broader than underlying policies                │
  │ • Drop-down provision for claims not covered by          │
  │   underlying (subject to self-insured retention)         │
  │ • Own set of definitions and exclusions                  │
  │ • Defense costs may be WITHIN limits (eroding)           │
  │                                                          │
  │ Example:                                                 │
  │ Primary CGL: $1M per occurrence                          │
  │ Umbrella: $5M xs $1M primary                             │
  │ SIR: $10,000 for drop-down coverage                     │
  │ → Umbrella pays $1M to $6M                               │
  │ → Drop-down: pays claims beyond CGL scope (SIR applies)  │
  └──────────────────────────────────────────────────────────┘

  TOWER OF COVERAGE:
  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │  $25M ┌────────────────────────────┐                    │
  │       │ 3rd Excess: $10M xs $15M   │                    │
  │  $15M ├────────────────────────────┤                    │
  │       │ 2nd Excess: $5M xs $10M    │                    │
  │  $10M ├────────────────────────────┤                    │
  │       │ 1st Excess/Umbrella:       │                    │
  │       │ $5M xs $5M                 │                    │
  │  $5M  ├────────────────────────────┤                    │
  │       │ Primary Layer:             │                    │
  │       │ CGL: $1M per occ           │                    │
  │       │ Auto: $1M CSL              │                    │
  │       │ Employers: $1M per occ     │                    │
  │  $1M  │ (Schedule of Underlying)   │                    │
  │       │ Gap: $4M SIR               │                    │
  │  $0   └────────────────────────────┘                    │
  └──────────────────────────────────────────────────────────┘
```
