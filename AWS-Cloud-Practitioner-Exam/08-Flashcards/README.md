# Flashcards — 600+ Cards

Each card is written as `Q: …` on one line, `A: …` on the next. Tags at
the end (`#domain2 #iam #must-know`) help you filter for spaced repetition.

If you import these into Anki or Mochi, the format translates to
"Basic" cards with no further work (see `import-guide.md`).
