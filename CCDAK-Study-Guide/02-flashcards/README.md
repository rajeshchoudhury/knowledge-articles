# Flashcards

500+ Q&A pairs for spaced-repetition review. Cover every topic that appears on the CCDAK exam.

## Suggested Workflow

1. Pick one deck per day during Weeks 1–6.
2. First pass: read Q and A in sequence.
3. Subsequent passes: read Q only, say A aloud, then reveal.
4. Mark any card you get wrong; re-drill until 100%.
5. Week 8: re-run every deck once.
