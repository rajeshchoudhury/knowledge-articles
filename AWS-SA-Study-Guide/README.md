## Exam Tips

1. **Eliminate obviously wrong answers** - Usually 1-2 answers can be eliminated immediately
2. **Look for keywords** - "cost-effective", "most secure", "least operational overhead", "highly available"
3. **Understand the Well-Architected Framework** - Many questions map to its pillars
4. **Time management** - ~2 minutes per question, flag and return to difficult ones
5. **Read the FULL question** - Requirements are often in the last sentence
6. **Default to AWS managed services** - AWS prefers their managed solutions
7. **Security is always important** - When in doubt, the more secure option is usually correct
8. **Multi-AZ ≠ Multi-Region** - Know the difference and when each is appropriate
